🎨 ランダムカラーボックス Math.random + style操作
📖 どう動く？ ボタンを押すと16進数のランダムなカラーコードを生成し、ボックスのbackgroundColorに適用。
🧠 使っているJS: Math.random()で数値生成 → 16進数変換 → style.backgroundColor で動的にスタイル変更。 
// ランダム色生成の核心部分
const box = document.getElementById('colorBox');
const randomColorBtn = document.getElementById('randomColorBtn');

randomColorBtn.addEventListener('click', () => {
  const randomColor = '#' + Math.floor(Math.random() * 16777215).toString(16).padStart(6, '0');
  box.style.backgroundColor = randomColor;
});
🎲 16777215 = 0xFFFFFF、ランダムな色の生成ロジックをマスター！
👻 表示・非表示トグル classList / display制御
🧙‍♂️ 「ここに隠れているメッセージだよ！ 見えたり消えたりできる！」
📖 どう動く？ ボタンを押すたびに、テキスト要素に対して.hiddenクラスを付け外し。CSSでdisplay: noneを制御。
🧠 重要コード: classList.toggle('hidden') で表示状態を簡単に切り替え。 
// トグルギミック
const textBlock = document.getElementById('toggleText');
const toggleBtn = document.getElementById('toggleVisibilityBtn');

toggleBtn.addEventListener('click', () => {
  textBlock.classList.toggle('hidden');
});

/* CSSで .hidden { display: none; } を定義済み */
💡 classList.toggle は条件分岐を書かなくてOK。便利！
📝 ライブテキスト反映 inputイベント
✨ 入力した文字がここに反映されます ✨
📖 どう動く？ 入力フィールドに文字を打つたび（または削除）に、下のプレビューエリアがリアルタイム更新。
🧠 キーコード: inputイベントを使い、event.target.valueを取得 → textContentで表示。 
const inputField = document.getElementById('liveInput');
const outputSpan = document.getElementById('liveOutput');

inputField.addEventListener('input', (event) => {
  const currentText = event.target.value;
  outputSpan.textContent = currentText || '✨ 入力した文字がここに反映されます ✨';
});
🔄 inputイベントはキーボードだけでなくペーストなど全ての変更をキャッチ
🪟 モーダルウィンドウ class追加/削除 & イベント伝搬
※ 背景クリックでも閉じられます
📖 どう動く？ ボタンでオーバーレイにactiveクラス追加 → モーダル出現。閉じるボタン or 背景クリックでクラス削除。
🧠 重要なテクニック: classList.add/remove とイベント伝搬の制御。 
// モーダル制御
const modalOverlay = document.getElementById('modalOverlay');
const openBtn = document.getElementById('openModalBtn');
const closeModalBtn = document.getElementById('closeModalBtn');

openBtn.addEventListener('click', () => modalOverlay.classList.add('active'));
closeModalBtn.addEventListener('click', () => modalOverlay.classList.remove('active'));
modalOverlay.addEventListener('click', (e) => {
  if(e.target === modalOverlay) modalOverlay.classList.remove('active');
});
✨ オーバーレイ自体をクリックした時だけ閉じる条件分岐がポイント
🐭 ホバーエフェクト mouseenter / mouseleave
😊
👆 アイコンにマウスを乗せる (タップ長押しでも変化)
📖 どう動く？ 絵文字エリアにマウスが入ると「🎉」に変わり、離れると元の「😊」に戻る。
🧠 使っているイベント: mouseenter と mouseleave でコンテンツのテキストを差し替え。 
const icon = document.getElementById('hoverIcon');

icon.addEventListener('mouseenter', () => {
  icon.textContent = '🎉';
});
icon.addEventListener('mouseleave', () => {
  icon.textContent = '😊';
});

// モバイルでもタップ＆ホバー擬似的に動作（タップ時にenter/leaveの代替）
💡 ホバーだけでなくタッチデバイスでも触ると変化、実用的なインタラクション
